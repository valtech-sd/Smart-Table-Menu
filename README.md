# Smart-Table-Menu

Demo of enabling a user to make a selection from a menu projected onto a table using hand gestures

## Getting Started

1. Install the dependencies

```
pnpm install
```

2. Create a `.env.local` file with the following values

```txt
VITE_FLIPPED_VIDEO=false
```

3. Run the project

```
pnpm run dev
```
